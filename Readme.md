# autotrader-query

[script]

query the autotrader API (to search for "good" openpilot-compatible cars :D)

### Prereqs

- ruby

### Install

    bundle

#### Run

    rake

#### Notes

This script will visit 2-3 pages of the autotrader website and pull the cheapest cars it can find

The output format is YAML

Don't misuse please :)
